Ласкаво просимо до Графічного Редактора

Ознайомтесь з інструментами перед початком малювання

Коло

Малює фігуру з рівномірним радіусом навколо центральної точки. Ідеально для створення кругів.

Квадрат

Фігура з рівними сторонами та кутами. Підходить для побудови рівномірних форм.

Прямокутник

Форма з чотирма кутами, у якій протилежні сторони рівні. Універсальний інструмент.

Пряма

Малює лінію між двома точками. Використовується для контурів і з'єднань.

Олівець

Вільне малювання по полотну. Слідує за рухами миші в реальному часі.

Стирачка

Видаляє частини малюнку, імітуючи біле перо. Корисна для корекцій.
